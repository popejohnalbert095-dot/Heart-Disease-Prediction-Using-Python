#  Heart Disease Prediction using Python

##  Project Overview
This project analyzes a Heart Disease dataset using Python. It performs data preprocessing, exploratory data analysis (EDA), and builds a Machine Learning model to predict whether a patient has heart disease.

---

##  Features

Load Heart Disease dataset
Dataset overview and statistical summary
Data cleaning
  - Missing value check
  - Duplicate removal
Exploratory Data Analysis (EDA)
Data visualization
Correlation analysis
Logistic Regression model
Model accuracy evaluation
Heart disease prediction for a new patient
Save cleaned dataset
